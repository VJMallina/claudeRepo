üîê Authentication (11 endpoints)

POST /auth/register

Register a new user with mobile number

Request: { mobileNumber: string }
Response: { userId: string, otpSent: boolean }
POST /auth/verify-otp

Verify OTP for registration or login

Request: { userId: string, otp: string }
Response: { accessToken: string, refreshToken: string }
POST /auth/login/pin

Login with PIN code

Request: { mobileNumber: string, pin: string }
Response: { accessToken: string, refreshToken: string }
POST /auth/login/otp

Request OTP for login

Request: { mobileNumber: string }
Response: { otpSent: boolean }
POST /auth/refresh-token

Refresh access token using refresh token

Auth: Bearer Token
Response: { accessToken: string }
GET /auth/me

Get current authenticated user details

Auth: Bearer Token
Response: User object with profile and KYC status
POST /auth/setup-pin

Setup or update PIN for quick login

POST /auth/enable-biometric

Enable biometric authentication

POST /auth/logout

Logout and invalidate tokens

POST /auth/forgot-pin

Reset PIN via OTP verification

POST /auth/change-mobile

Change registered mobile number

‚úÖ KYC Verification (9 endpoints)

GET /kyc/status

Get current KYC level and completion status

Auth: Bearer Token
Response: { level: 0|1|2, status: string, completedSteps: [] }
POST /kyc/verify-pan

Verify PAN card with Income Tax Department

Request: { panNumber: string, name: string, dob: string }
Response: Verification result and KYC level update
POST /kyc/verify-aadhaar/initiate

Initiate Aadhaar verification (sends OTP)

Request: { aadhaarNumber: string }
Response: { requestId: string, otpSent: boolean }
POST /kyc/verify-aadhaar/complete

Complete Aadhaar verification with OTP

Request: { requestId: string, otp: string }
Response: Verified Aadhaar details and address
POST /kyc/verify-bank

Verify bank account with penny drop test

Request: { accountNumber: string, ifsc: string, name: string }
POST /kyc/verify-liveness

Face liveness detection for Level 2 KYC

Request: Image/video file for face verification
Response: Liveness check result and face match score
POST /kyc/upload-document

Upload KYC supporting documents

GET /kyc/documents

Get list of uploaded KYC documents

GET /kyc/limits

Get transaction limits based on KYC level
